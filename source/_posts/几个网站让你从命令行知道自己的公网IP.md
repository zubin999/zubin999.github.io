- ifconfig.me
- ipinfo.io/ip
- api.ipify.org
- ipecho.net/plain
- wtfismyip.com/json

``` shell
curl ifconfig.me
curl ipinfo.io/ip
curl api.ipify.org
curl ipecho.net/plain
curl wtfismyip.com/json
```
